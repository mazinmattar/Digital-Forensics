## **Live Acquisition vs Dead Acquisition**

1. Live Acquisition
→ يعني بناخد الأدلة والجهاز لسه شغال.

زي:

- الرام
- العمليات اللي شغالة
- الاتصالات الحالية
- اليوزرز اللي فاتحين الجهاز

مميزاته:

- بنلحق ناخد الـ Volatile Data قبل ما تضيع.

عيوبه:

- ممكن وإنت شغال تغيّر جزء بسيط من الأدلة.
1. Dead Acquisition
→ يعني بناخد الأدلة بعد ما الجهاز يقفل.

بنركز على:

- الهارد
- الملفات
- الـ Metadata
- الملفات المحذوفة

مميزاته:

- أكثر أمانًا على الأدلة.
- بيقلل احتمالية تغيير البيانات.

عيوبه:

- الرام والاتصالات الحالية بتضيع.
