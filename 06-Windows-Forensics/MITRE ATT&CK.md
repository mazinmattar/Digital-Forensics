Framework بيصنف أساليب وتقنيات المهاجمين المستخدمة أثناء الهجمات.

بيساعد الـ Blue Team وDFIR Analysts يفهموا مراحل الهجوم زي:

- Initial Access
- Persistence
- Privilege Escalation
- Defense Evasion
- Credential Access
- Discovery
- Lateral Movement
- Command and Control
- Exfiltration

أهمية MITRE ATT&CK:

- فهم طريقة تفكير المهاجم
- ربط الـ Artifacts بالهجمات
- تحسين الـ Detection والـ Threat Hunting
