هو تحليل الترافيك والاتصالات الشبكية علشان نكتشف أي نشاط مشبوه أو هجمات حصلت على الشبكة.

الحاجات اللي بنحللها:

- PCAP Files
- DNS Traffic
- HTTP Requests
- IP Connections
- Network Logs

أشهر الأدوات:

- Wireshark
- tcpdump
- Zeek
- NetworkMiner

الحاجات المشبوهة اللي بندور عليها:

- اتصالات خارجية غريبة
- Data Exfiltration
- DNS Tunneling
- Beaconing
- Command and Control (C2)

أمثلة مهمة:

- تحليل الـ Source و Destination IP
- معرفة الـ Open Ports
- فحص الـ User-Agent
- تحليل الـ HTTP Methods
- اكتشاف الاتصالات المتكررة المشبوهة

-Indicators of Compromise (IOCs)

هي علامات أو آثار بتدل إن الجهاز أو الشبكة ممكن يكونوا اتعرضوا لهجوم.

أمثلة على الـ IOCs:

- IPs مشبوهة
- Domains غريبة
- Hashes لملفات Malware
- اتصالات خارجية غير معتادة
- Processes مشبوهة
- PowerShell Commands غريبة
- Beaconing Traffic

أهمية الـ IOCs:
بتساعد الـ Blue Team والـ DFIR Analysts في اكتشاف الاختراقات بسرعة وتتبع نشاط المهاجم.
